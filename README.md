 CS50x - Week 0: Bug Simulator Game

Welcome to my first project in CS50! This is a 2D survival/strategy game built using **Scratch** to demonstrate basic programming logic like loops, variables, and clones.

 Game Concept
You play as a stressed programmer trying to finish your CS50 PSET before the deadline, while fighting off annoying system **Bugs** that try to crash your computer and drain your mental health!

 Features Implemented
- Dynamic Variable System: Tracks Code Progress, Mental Health, and Coffee inventory.
- Cloning System: Random bugs spawn and move toward the desk dynamically.
- Interactive UI: Click bugs to destroy them, or click coffee to restore energy.

 How to Play
1. Mash the `Space` bar to write code and increase your progress.
2. Click on incoming bugs quickly before they hit the computer.
3. Don't let your `Mental Health` drop to 0%!
